# Deployment Guide - Campus Events Hub

## Prerequisites
- GitHub account (already have repo)
- Vercel account (free)
- Render account (free)
- MongoDB connection string (already have)

---

## PART 1: PREPARE FOR DEPLOYMENT

### Step 1: Merge feature branch to main
```bash
git checkout main
git merge fix/venue-token-key
git push origin main
```

### Step 2: Verify environment variables are set

**Backend (.env):**
```
MONGODB_URI=mongodb://...your_connection_string...
JWT_ACCESS_SECRET=event_mgmt_access_secret_dev_key_2024
JWT_REFRESH_SECRET=event_mgmt_refresh_secret_dev_key_2024
PORT=4000
NODE_ENV=production
```

**Frontend (.env.production)** - Create this file in frontend folder:
```
VITE_API_URL=https://YOUR_RENDER_APP.onrender.com/api
```

---

## PART 2: DEPLOY BACKEND TO RENDER

### Step 1: Create Render Account
1. Go to https://render.com
2. Sign up with GitHub
3. Click "New +" → "Web Service"
4. Select your GitHub repo `campus-events-hub`
5. Configure as follows:

**Settings:**
- **Name**: `campus-events-hub-api`
- **Environment**: `Node`
- **Region**: Choose closest to you
- **Branch**: `main`
- **Build Command**: `cd backend && npm install && npm run build`
- **Start Command**: `cd backend && npm start`

### Step 2: Add Environment Variables in Render
In the Render dashboard:
1. Go to your Web Service
2. Click "Environment"
3. Add these variables:
   ```
   MONGODB_URI = mongodb://parikshithsr_demo:1122334455@...
   JWT_ACCESS_SECRET = event_mgmt_access_secret_dev_key_2024
   JWT_REFRESH_SECRET = event_mgmt_refresh_secret_dev_key_2024
   PORT = 4000
   NODE_ENV = production
   ```

### Step 3: Deploy
Click "Create Web Service" - Render will auto-deploy from main branch

**Backend URL will be**: `https://campus-events-hub-api.onrender.com`

---

## PART 3: DEPLOY FRONTEND TO VERCEL

### Step 1: Create Vercel Account
1. Go to https://vercel.com
2. Sign up with GitHub
3. Click "Add New..." → "Project"
4. Import your `campus-events-hub` repository

### Step 2: Configure Project
In Vercel dashboard:

**Settings:**
- **Framework Preset**: Vite
- **Root Directory**: `frontend`
- **Build Command**: `npm run build`
- **Output Directory**: `dist`
- **Install Command**: `npm install`

### Step 3: Add Environment Variables
In Vercel project settings → "Environment Variables":

Add:
```
VITE_API_URL = https://campus-events-hub-api.onrender.com/api
```

### Step 4: Deploy
Click "Deploy" - Vercel will auto-deploy from main branch

**Frontend URL will be**: `https://campus-events-hub.vercel.app` (auto-generated or custom domain)

## TROUBLESHOOTING

### Backend Won't Start on Render
- Check "Logs" tab in Render dashboard
- Verify MongoDB connection string is correct
- Ensure all environment variables are set

### Frontend Shows "API Connection Error"
- Verify `VITE_API_URL` environment variable in Vercel
- Check CORS settings in backend `app.ts`
- Verify backend URL is correct

### Venues Not Loading
- Check browser console for network errors
- Verify JWT tokens are being sent correctly
- Ensure collegeId is included in JWT payload

### Database Issues
- Verify MongoDB connection string
- Check IP whitelist in MongoDB Atlas
- Ensure database and collections exist
